## 作品简介

1. 这个是一个用于学习Minecraft的运行机制而开发的项目，
2. 它是使用Java语言编写的，基于Minecraft 1.8.8。
3. 它是一个Minecraft PVP客户端。
4. 可以将它用于学习Minecraft的运行机制，但不可以用于商业用途。
5. 目前的版本是v1.0.0_alpha。

## 运行

1. 安装Java 8。
2. 下载并解压本仓库。
3. 使用JetBrains的IntelliJ IDEA打开项目。
4. 在IDEA里运行应用程序：StartSC
